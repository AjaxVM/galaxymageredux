INSTALLATION

Redux is written in Python, so you will need a Python interpreter to run it.
You will also need a couple of Python libraries.

* Python
  http://python.org

* Pyglet
  http://pyglet.org

* Twisted
  http://twistedmatrix.com

After everything is installed properly you should be able run the game by
typing

$ python Redux.py

into a terminal or by double clicking Redux.py in a graphical file manager.
